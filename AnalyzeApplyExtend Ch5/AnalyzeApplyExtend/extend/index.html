Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

position: sticky; is like a relative element until it reaches a specified point, and then it acts like a fixed element. So, it moves with the page until it reaches a certain point, and then it stays in that position as you continue scrolling.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

If you move a relative element, it affects the layout of other elements around it, but it doesn't leave an empty space where it was.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

A fixed element is positioned relative to the viewport, so it stays in the same place even when you scroll the page.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

position: absolute; is like telling an element to find its "parent" element that has a position value of relative, absolute, or fixed. Once it finds that parent, it positions itself inside that parent element according to the top, bottom, left, and right properties you specify.
